# 📊 Superstore Sales Dashboard | Power BI

A modern and interactive Business Intelligence dashboard built using **Power BI Desktop** and the **Superstore Sales Dataset**.
This project focuses on transforming raw sales data into meaningful business insights through data cleaning, modeling, visualization, and dashboard storytelling.

---

# 🚀 Project Overview

The goal of this project is to analyze and visualize sales performance across different regions, categories, customer segments, and time periods.

The dashboard helps answer important business questions such as:

* Which category generates the highest sales?
* Which region produces the highest profit?
* How are sales trending over time?
* Which customer segment contributes most revenue?
* Which sub-categories drive the business?

This dashboard was designed with a professional corporate UI inspired by modern BI dashboards used in real companies.

# 📂 Dataset Information

**Dataset:** Sample Superstore Sales Dataset
**Source:** Kaggle
**Rows:** ~9,994
**Format:** CSV

Dataset contains:

* Orders
* Sales
* Profit
* Discounts
* Categories
* Regions
* Customer Segments
* Shipping Information

---

# 🔄 Data Cleaning & Transformation

The dataset was cleaned and transformed using **Power Query Editor**.

### Transformations Performed:

* Removed unnecessary columns
* Filtered customer segments
* Removed "Home Office" segment
* Changed incorrect data types
* Split `Order ID` using delimiter
* Renamed columns for readability
* Removed null values
* Optimized dataset structure

---

# 📈 KPI Metrics Created

The following business KPIs were created using DAX measures:

* Total Sales
* Total Profit
* Total Quantity
* Average Discount
* Profit Margin %
